# E-Book-Rat

## Description
This is a simple python script that will download all the books from the website [https://www.gutenberg.org/](https://www.gutenberg.org/). It will download all the books in the format of .txt and will save it in the folder named "Books". It will also create a folder named "Books" if it doesn't exist.
